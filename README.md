# 💻 Intelligent IT Support & Troubleshooting Portal

A full-featured IT Help Desk simulation built with **Python, Streamlit, and SQLite**, designed to demonstrate real-world IT support workflows including troubleshooting, ticket management, SLA tracking, and dashboard analytics.

---

## 🚀 Features

### 👤 User Features

* Guided troubleshooting for common IT issues
* Searchable Knowledge Base
* Support ticket creation with attachments
* Ticket tracking via *My Tickets*
* Comment-based communication with IT support
* Notifications for ticket updates

---

### 👨‍💼 Admin Features

* Dashboard with analytics and insights
* Ticket lifecycle management (Open → In Progress → Resolved → Closed)
* SLA tracking and priority handling
* Ticket assignment and escalation
* Activity timeline (audit trail)
* Resolution templates for faster handling
* Knowledge Base management
* CSV export of tickets
* Demo data tools for testing

---
