# 🧠 Intelligent IT Support & Troubleshooting System

An intelligent IT support application designed to simulate real-world helpdesk workflows.  
The system combines a structured Knowledge Base, guided troubleshooting, and a full ticket management system for both users and administrators.

---

## 🚀 Live Demo
👉 https://it-support-troubleshooting-app.streamlit.app/

---

## 🎯 Project Overview

This project was built to demonstrate practical IT support skills, including:
- troubleshooting workflows
- incident management
- user vs admin interaction
- issue prioritization and diagnosis

It reflects how real IT helpdesk systems operate.

---

## 🧩 Features

### 👤 User Side
- Create support tickets
- Guided troubleshooting suggestions
- User-friendly troubleshooting steps
- View personal tickets ("My Tickets")
- Upload attachments (screenshots, logs)
- Receive notifications when admin replies

---

### 👨‍💼 Admin Side
- View and manage all tickets
- Priority detection (Low → Critical)
- Assign, update, and resolve tickets
- Add comments (communication with users)
- View attachments (image preview + downloads)
- Dashboard with ticket statistics
- Detect infrastructure-wide issues
- Manage Knowledge Base

---

### 📚 Knowledge Base
- Categorized IT issues (Network, Email, System, etc.)
- Search & filtering
- Symptoms, causes, and solutions
- Role-based troubleshooting:
  - User-friendly steps
  - Advanced IT steps

---

### 🔔 Smart Features
- Suggest issues based on user input
- Detect possible infrastructure incidents
- Comment notifications (user ↔ admin)
- Role-based experience

---

## 🏗 Architecture

- **Frontend:** Streamlit
- **Backend:** Python
- **Database:** SQLite (`it_support.db`)
- **File Storage:** Local (attachments)

---

## 🧠 Skills Demonstrated

- IT troubleshooting methodology
- Helpdesk workflow design
- User experience (technical vs non-technical users)
- Incident management logic
- Backend data handling with SQLite
- Application structuring and modular design

---

## ⚠️ Notes

- This is a demonstration project for portfolio purposes
- Authentication is simplified (no encryption for passwords)
- File storage is local (not cloud-based)
